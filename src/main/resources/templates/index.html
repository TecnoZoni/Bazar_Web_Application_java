Bienvenido al Bazar

Administra clientes, ventas y productos de manera fácil y rápida.

Descripción del Programa

El sistema de gestión del bazar permite administrar clientes, productos y ventas a través de una API Restful. Las operaciones que se pueden realizar son Crear, Leer, Actualizar y Eliminar (CRUD) usando solicitudes HTTP.

Métodos HTTP Configurados

GET

Para obtener la lista de todos los recursos o uno específico.

  • /cliente/verClientes para ver todos los clientes.
  • /producto/verProductos para ver todos los productos.
  • /venta/verVentas para ver todos los ventas.
  • /cliente/verCliente/{id} para ver un cliente por ID.
  • /producto/verProducto/{id} para ver un producto por ID.
  • /venta/verVenta/{id} para ver una venta por ID.
POST

Para crear nuevos recursos en el sistema.

  • /cliente/crearCliente para crear un nuevo cliente.
  • /producto/crearProducto para crear un nuevo producto.
  • /venta/crearVenta para crear una nueva venta.
PUT

Para actualizar un recurso existente.

  • /cliente/editarCliente/{id} para editar un cliente.
  • /producto/editarProducto/{id} para editar un producto.
  • /venta/editarVenta/{id} para editar una venta.
DELETE

Para eliminar un recurso existente.

  • /cliente/eliminarCliente/{id} para eliminar un cliente.
  • /producto/eliminarProducto/{id} para eliminar un producto.
  • /venta/eliminarVenta/{id} para eliminar una venta.

Ejemplos de Objetos JSON

Cliente

Cuando creas o actualizas un cliente, el servidor espera un objeto JSON como el siguiente:

{
    "nombre": "Juan",
    "apellido": "Pérez",
    "dni": "12345678"
}
Producto

Para la creación o actualización de un producto, se espera un objeto JSON como el siguiente:

{
    "nombre": "Taza",
    "marca": "Cerámicas S.A.",
    "costo": 250.00,
    "cantidad_disponible": 100
}
Venta

Al crear o editar una venta, el servidor espera un objeto JSON que incluya un cliente y una lista de productos:

{
    "fecha_venta": "2024-10-18",
    "total": 500.00,
    "unCliente": {
        "id_cliente": 1
    },
    "listaProductos": [
        {
            "codigo_producto": 101
        },
        {
            "codigo_producto": 102
        }
    ]
}
